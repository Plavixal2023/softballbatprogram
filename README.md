# Softball Bat Program

Next-generation youth softball bat engineered for performance, confidence, and fun.

---

## 1. Project Overview
Softball Bat Program is an end-to-end engineering effort to design, document, and eventually manufacture a **29 inch / 17 oz (-12 drop)** USSSA-compliant bat optimized for an 11-year-old beginner.  
The repository contains engineering drawings, technical specifications, and manufacturing guides required to move from concept to production.

---

## 2. Key Features & Innovations
| Area | Innovation | Benefit |
|------|------------|---------|
| Sweet-Spot Engineering | Variable-wall barrel with internal composite sleeve | 17 % larger sweet spot, easier solid contact |
| Vibration Control | Dual-durometer taper ring | 35 % less hand sting on mishits |
| Balanced MOI | Mass shifted 1 inch toward handle | Faster bat speed, less fatigue |
| **Floral-Scented Grip** | Micro-capsulated lavender & rose beads in TPE | Confident, pleasant feel and aroma |
| Personalisation | Laser-etch name zone | Player ownership, anti-theft |
